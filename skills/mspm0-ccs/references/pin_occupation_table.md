# MSPM0G3519 引脚占用表

## 调试接口

| 引脚 | 功能 | 外设 | 占用情况 | 备注 |
|------|------|------|----------|------|
| PA19 | SWDIO | SWD | 调试占用 | **不可使用**，不能连接其他设备 |
| PA20 | SWCLK | SWD | 调试占用 | **不可使用**，不能连接其他设备 |

## 通信接口

| 引脚 | 功能 | 外设 | 占用情况 | 备注 |
|------|------|------|----------|------|
| PA10 | TX0 | UART0 | 板载 CH340 | 可通过 TYPE-C 或下载器接口调试，排针可外接模块（互不冲突） |
| PA11 | RX0 | UART0 | 板载 CH340 | 可通过 TYPE-C 或下载器接口调试，排针可外接模块（互不冲突） |
| PA0 | SDA | I2C0 | OLED + 六轴陀螺仪 | 板载 2.2K 上拉电阻；I2C 总线支持多设备，可继续外接其他 I2C 从设备 |
| PA1 | SCL | I2C0 | OLED + 六轴陀螺仪 | 板载 2.2K 上拉电阻；I2C 总线支持多设备，可继续外接其他 I2C 从设备 |
| PB18 | RX | UART7 | 无线串口模块 | |
| PB17 | TX | UART7 | 无线串口模块 | |
| PB6 | CS | SPI1_CS0（GPIO 方式） | 外部 Flash W25Q64 | 通过普通 GPIO 控制，未配置到 SPI 外设 |
| PB7 | MISO (POCI) | SPI1_POCI | 外部 Flash W25Q64 | |
| PB8 | MOSI (PICO) | SPI1_PICO | 外部 Flash W25Q64 | |
| PB9 | SCLK | SPI1_CLK | 外部 Flash W25Q64 | |

## 板载外设

| 引脚 | 功能 | 外设 | 占用情况 | 备注 |
|------|------|------|----------|------|
| PB26 | WS2812 RGB 灯 | TIMA1_C0（PWM） | 板载 4 个 RGB 灯 | 单总线控制 |
| PB27 | 无源蜂鸣器 | TIMG8_C1（PWM） | 板载蜂鸣器 | PWM 控制频率和音量 |
| PB22 | LED | GPIO | 板载 LED | 高电平灭，低电平亮 |
| PA29 | 编码器 A 相 | QEI（IIMG8_C0/C1） | 旋转编码器（可选） | 如不使用旋转编码器，可释放用作其他功能 |
| PA30 | 编码器 B 相 | QEI（IIMG8_C0/C1） | 旋转编码器（可选） | 如不使用旋转编码器，可释放用作其他功能 |
| PA31 | 编码器 S（按下） | GPIO | 旋转编码器（可选） | 按下检测；如不使用旋转编码器，可释放用作其他功能 |

## 系统/按键

| 引脚 | 功能 | 外设 | 占用情况 | 备注 |
|------|------|------|----------|------|
| PA18 | BSL 按键 | GPIO | 系统/按键 | 高电平按下；同时也是 BSL 模式引脚，**正常执行程序时必须确保不为高电平** |
| PB21 | 按键2 | GPIO | 用户按键 | 低电平按下，用户可自定义 |
| PB23 | 连接状态检测 | GPIO | 无线串口模块 | 低电平 = 连接成功，高电平 = 失败 |

## 系统级不可用引脚

| 引脚 | 功能 | 外设 | 占用情况 | 备注 |
|------|------|------|----------|------|
| PA2 | 频率精度控制 | - | 系统占用 | **不可使用**，不能连接其他设备，未引出 |
| PA5 | 晶振 | - | 系统占用 | **不可使用**，不能连接其他设备 |
| PA6 | 晶振 | - | 系统占用 | **不可使用**，不能连接其他设备 |

## 汇总

| 类别 | 引脚列表 |
|------|----------|
| **完全不可用（系统占用）** | PA2, PA5, PA6, PA19, PA20 |
| **有条件不可用** | PA18（需确保不为高电平才能正常执行程序） |
| **已占用但可共用总线** | PA0, PA1（I2C0，支持多设备，可继续外接 I2C 从设备） |
| **已占用但排针可共用** | PA10, PA11（UART0，排针可外接） |
| **可选外设（不使用时可释放）** | PA29, PA30, PA31（旋转编码器） |
| **已占用（板载设备）** | PB6, PB7, PB8, PB9, PB17, PB18, PB22, PB23, PB26, PB27, PB21 |

> **注意**：其余未列出的引脚均已引出到排针，用户可自由使用。具体外设支持情况请参考本地 `外设引脚功能标注表.xlsx`，其中黄色背景单元格表示不可使用。
