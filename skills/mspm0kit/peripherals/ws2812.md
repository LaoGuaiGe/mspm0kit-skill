# WS2812 RGB LED on Tianqiaoxing G3519

**Driver**: `hardware/hw_ws2812.c/h`
**Effects**: `hardware/hw_ws2812_effects.c/h`

## Hardware

- LED: 4× WS2812 addressable RGB LEDs
- Control: TIMA1 CCP0 PWM on PB26
- Protocol: Single-wire (800 kHz NZR), 80 MHz clock, period 100 counts

## Adding to OLED UI project

```bash
python scripts/scaffold_oled.py <name> --with-ws2812
```

This copies: `hw_ws2812.c/h`, `hw_ws2812_effects.c/h`, `mid_timer_stub.c/h`

And adds to `.syscfg`: TIMA1 PWM on PB26 + TIMA0 5ms timer.

## Generated Macros (from syscfg)

```
WS2812_INST          → TIMA1
GPIO_WS2812_C0_PIN   → DL_GPIO_PIN_26
GPIO_WS2812_C0_IDX   → DL_TIMER_CC_0_INDEX
WS2812_INST_CLK_FREQ → 80000000
TIMER_TICK_INST      → TIMA0
```

## Key APIs

```c
#include "hw_ws2812.h"
#include "hw_ws2812_effects.h"

// Init
PWM_WS2812B_Init();

// Set single LED color (index 0-3, R/G/B 0-255)
WS2812B_SetPixel(0, 255, 0, 0);  // LED0 = red
WS2812B_Show();                   // Push to LEDs

// Effects (requires timer_init() and ws2812_effect_update() in loop)
ws2812_effect_tick();             // Call from timer ISR (via mid_timer_stub)
WS2812_SetMode(MODE_FLOW);       // MODE_SOLID / MODE_FLOW / MODE_MARQUEE
WS2812_SetBrightness(50);         // 0-100
WS2812_SetSpeed(30);              // 0-100
WS2812_SetLEDCount(4);            // 1-4

// In main loop:
while (1) {
    ws2812_effect_update();   // Render current frame
    OLED_UI_MainLoop();
}
```

## Effects

| Mode | Constant | Description |
|------|----------|-------------|
| Solid | `MODE_SOLID` | Fixed color on all LEDs |
| Flow | `MODE_FLOW` | Color flowing across LEDs |
| Marquee | `MODE_MARQUEE` | Color chasing pattern |

## Dependencies

- `hw_delay.c/h` (nop-based delay, already in base OLED project)
- `mid_timer_stub.c/h` (5ms tick for effect timing)
